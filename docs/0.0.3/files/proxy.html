    Very simple script to act as proxy to retrieve the Modernizr Documentation web page.

    This script acts as a proxy to retrieve the Modernizr Documentation. The checks in this script are simple and fallable checks to prevent misuse. As the scenarios for misuse are rather limited (as in: I can't imagine this script to be misused in any way, but maybe I lack imagination), I guess these simple checks should suffice. Hey, at least I have considered security and that's more than you can say of most IoT manufacturers...
