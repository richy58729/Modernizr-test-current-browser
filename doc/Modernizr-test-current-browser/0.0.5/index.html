Modernizr-test-current-browser 0.0.5

This module shows a table with all the features Modernizr provides where each cell indicates if the current browser supports the feature or not.

If documentation about the feature could be distilled from the Modernizr Documentation, it will be shown when hovering the mouse cursor over the table row of the feature. This doesn't work 100% flawlessly, but most of the time I think it's quite handy despite this.

Currently uses Modernizr v3.6.0.
Uses DataTables for formatting the features in a handy table with search capability and pagination.
