doc :String

Document that has been retrieved by getting the contents of the Modernizr Documentation page at https://modernizr.com/docs and then DOMified by means of the jQuery parseHTML function. Because that page is on another domain, simply getting it by means of AJAX wasn't an option, so this page is retrieved by means of a (very) simple PHP script that acts as proxy. Potentially that could be harmful if the Modernizr Documentation page would have been injected with malware, but as it's kept in a separate variable instead of being injected in the current document's DOM, I think the risk is really low to non-existent.
